# Task Manager (PHP + MySQL)

## Overview
A simple Task Manager application built using PHP & MySQL.  
Supports adding and deleting tasks using a clean UI.  
This project is designed to demonstrate full-stack development and prepare for DevOps (Docker, CI/CD, AWS).

## Local Setup (XAMPP)
1. Start Apache & MySQL in XAMPP Control Panel.
2. Place project in: C:\xampp\htdocs\task-manager\
3. Import init.sql in phpMyAdmin (creates database taskdb).
4. Open in browser: http://localhost/task-manager/

## Features
- Add Task
- Delete Task
- MySQL Database Integration
- Simple PHP Backend
